### System Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel Mac

## Features

### Dual Interface Design
Access your clipboard history two ways:
- **Menu bar overlay** — Quick access to recent items from your status bar
- **Full application window** — Complete history with advanced search and filtering

### Multiple Content Types
Cliparoo handles all your clipboard content:
- Text (with automatic URL detection)
- Images (with previews)
- URLs and links
- Files
- Rich Text (RTF)
- HTML content

### Smart Search & Organization
- Powerful search to find items instantly
- Automatic content categorization
- History of up to 1,000 items

### Keyboard-First Design
- Global hotkeys for quick access
- Full keyboard navigation
- Accessibility support built-in

## Privacy First

Cliparoo is designed for users who value their privacy:

- **100% local storage** — All clipboard data stays on your Mac
- **No cloud sync** — Your data never leaves your device
- **No telemetry** — No usage tracking or data collection
- **No account required** — Just download and use

Perfect for developers, security professionals, and anyone handling sensitive information.

## Installation

1. Download the DMG from [cliparoo.app](https://www.cliparoo.app) or [GitHub Releases](https://github.com/stwelgemoed/cliparoo-releases/releases/latest)
2. Open the DMG file
3. Drag Cliparoo to your Applications folder
4. Launch Cliparoo and grant the requested permissions

### Permissions

Cliparoo requires the following macOS permissions to function:
- **Accessibility** — For clipboard monitoring
- **Input Monitoring** — For global keyboard shortcuts
